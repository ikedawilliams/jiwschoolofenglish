AI Dojo

ひとりでも、
英語の練習はできる。

AI Dojoは、レッスンとレッスンの間に使える自主練ページです。ChatGPT、Gemini、ClaudeなどのAIにプロンプトをコピーして、会話・リスニング・発音・ライティングを練習できます。

How To Use

コピーして、少し変えて、練習する。

AIが初めてでも大丈夫です。まずは例をそのまま使い、慣れてきたら自分の目的に合わせて変えてください。

1

Part 1

最初に、あなたのレベル・目的・練習方法をAIに伝えます。

2

Part 2

練習したいメニューを1つだけ選んでコピーします。

3

Ask AI

わからない時は、そのままAIに聞けば大丈夫です。

Text Mode / Voice Mode

まずは、文字でも大丈夫。

AIに慣れていない人は、最初は文字で練習してOKです。慣れてきたら音声モードを使うと、会話・発音・リスニングの練習がしやすくなります。

Text Mode

文字で入力して、文字で返事をもらう練習です。AIが初めての人、声を出すのが不安な人、ライティング練習をしたい人におすすめです。

Voice Mode

文字を打つ代わりに、AIと声で話す機能です。ロールプレイ、発音、リスニング、会話練習に向いています。

Beginner Tip

わからない時は、AIに聞いてください。

AIは先生のように使えます。単語、文法、使い方、練習方法がわからない時は、そのまま質問してください。

  • この単語の意味を簡単に説明して。
  • もっと簡単な英語にして。
  • 日本語で説明して。
  • 例文を3つ見せて。
  • 私の文を自然な英語に直して。

Part 1

最初に送る自己設定

下は完成した例です。コピーしてから、名前・レベル・目的・練習方法を自分用に変えてください。Part 1では練習を始めません。AIにあなたの情報を伝えるだけです。

AI Prompt Example

自己設定最初にコピーするプロンプト

私の名前はケンジです。
私は英語を勉強している日本人です。
私の現在のレベルは初中級です。
英語を使う目的は海外旅行です。
練習方法は文字モードです。

私のレベルに合わせて、英語の難しさ、語彙、例文、修正の量、スピードを調整してください。
質問は1つずつしてください。
難しい言葉は、簡単に説明してください。
間違いがあれば、自然に短く直してください。
長い文法説明は、私がお願いした時だけしてください。

必要な情報が足りない場合は、練習を始める前に質問してください。
準備できたら、「OK、Part 2を送ってください。」とだけ返事してください。
自分用に変えるところ

ケンジ → 自分の名前 / 初中級 → 自分のレベル / 海外旅行 → 自分の目的 / 文字モード → 音声モードでもOK

Part 2

1つ選んで、練習する。

Part 2では、練習メニューを1つだけ選びます。レベル・目的・練習方法はPart 1で伝えてあるので、ここではもう書きません。

AI Prompt

ロールプレイ旅行・面接・日常会話

Part 1の内容を使ってください。
ロールプレイを始めてください。

場面はホテルチェックインです。
私はお客さんです。
あなたはホテルスタッフです。

ルール：
・役になりきってください。
・短く自然なやり取りにしてください。
・質問は1つずつしてください。
・私の返事の後、必要な時だけ短く1つ修正してください。
・その後、会話を続けてください。
・私がわからない時に使える「聞き返し表現」も自然に教えてください。

最初に場面を短く説明して、最初の質問をしてください。

AI Prompt

リスニング聞く練習

Part 1の内容を使ってください。
リスニング練習をしてください。

トピックは海外旅行です。
長さは30秒くらいにしてください。

ルール：
・最初はスクリプトを見せないでください。
・音声モードの場合は、英文を1回だけ読んでください。
・文字モードの場合は、「今から英文を出します」と言ってから英文を出してください。
・その後、理解チェックの質問を3つしてください。
・答え合わせの後に、同じ英文をもう一度出してください。
・最後に、難しい単語や表現を短く説明してください。

AI Prompt

発音自然な音・リズム

Part 1の内容を使ってください。
発音練習をしてください。

旅行でよく使う短い英文を練習したいです。

ルール：
・短い文を1つずつ出してください。
・発音、リズム、強く読むところを簡単に教えてください。
・音声モードの場合は、先にあなたが読んでください。
・私が繰り返した後、修正は1つだけにしてください。
・難しい専門用語や発音記号は使わないでください。

まず、簡単な文を1つ出してください。

AI Dojo

Practice alone.
Improve between lessons.

AI Dojo is a self-practice page for the time between lessons. Copy the prompts below into ChatGPT, Gemini, Claude, or another AI tool to practice speaking, listening, pronunciation, writing, grammar, and vocabulary.

How To Use

Copy. Change a little. Practice.

You do not need to understand AI perfectly. Start with the examples. When you feel comfortable, change them for your own goals.

1

Part 1

Tell the AI your level, goal, and practice method.

2

Part 2

Choose one training menu and copy it.

3

Ask AI

If you are confused, simply ask the AI to explain.

Text Mode / Voice Mode

Text is fine at first.

If you are new to AI, start with text. When you feel ready, use Voice Mode for speaking, pronunciation, listening, and role play.

Text Mode

You type to the AI and read the reply. This is good for beginners, shy learners, and writing practice.

Voice Mode

You speak to the AI instead of typing. This is useful for conversation, pronunciation, listening, and role play.

Beginner Tip

If you are confused, ask the AI.

AI can act like a practice partner and a simple teacher. If you do not understand a word, grammar point, instruction, or exercise, ask directly.

  • What does this word mean?
  • Can you explain that more simply?
  • Can you explain that in Japanese?
  • Can you give me three examples?
  • Can you correct my sentence naturally?

Part 1

Send your profile prompt first.

This is a complete example. Copy it first, then change the name, level, goal, and practice method. Part 1 does not start training. It tells the AI who you are.

AI Prompt Example

Profile PromptCopy this first

My name is Kenji.
I am a Japanese person studying English.
My current level is lower intermediate.
I want to improve my travel English.
I want to practice by text mode.

Please adjust your English level, vocabulary, examples, corrections, and speed to suit me.
Ask me one question at a time.
Explain difficult words simply.
Correct my mistakes naturally and briefly.
Do not give long grammar explanations unless I ask.

If you need more information, ask me before starting.
When you are ready, please respond only with: “OK, I am ready for Part 2.”
Change it for yourself

Kenji → your name / lower intermediate → your level / travel English → your goal / text mode → voice mode if you want to speak

Part 2

Choose one. Practice.

Choose only one training menu below. Your level, goal, and practice method are already in Part 1, so Part 2 does not ask for them again.

AI Prompt

Role PlayTravel / interview / daily life

Use my profile from Part 1.
Please start a role play.

The situation is hotel check-in.
I am the customer.
You are the hotel staff.

Rules:
- Stay in character.
- Use short, natural turns.
- Ask one question at a time.
- After my response, give one short correction only if needed.
- Then continue the conversation.
- If I do not understand, help me ask for repetition naturally.

Start now. Set the scene briefly and ask me the first question.

AI Prompt

ListeningUnderstanding practice

Use my profile from Part 1.
Please create a listening activity.

The topic is travel English.
The length should be about 30 seconds.

Rules:
- Do not show the script before the first listen.
- If we are using voice mode, read the script aloud once.
- If we are using text mode, say “Here is the listening text” and then show the text.
- Ask me 3 comprehension questions.
- After checking my answers, show the same text again.
- Finally, explain difficult words or phrases briefly.

AI Prompt

PronunciationSound and rhythm

Use my profile from Part 1.
Please help me practice pronunciation.

I want to practice short travel phrases.

Rules:
- Give me one short sentence at a time.
- Explain the pronunciation, rhythm, and stress simply.
- If we are using voice mode, say the sentence first.
- After I repeat, give only one correction.
- Do not use technical terms or IPA unless I ask.

Start with one easy sentence.

AI Prompt

WritingDiary / email / opinion

Use my profile from Part 1.
Please help me practice English writing.

First, ask me to choose one option:
1 Diary
2 Email
3 Short opinion
4 EIKEN-style answer

Rules:
- Give me one task at a time.
- Wait for me to write.
- Keep corrections short and clear.
- Show a more natural English version.
- Choose one sentence for me to rewrite.

Start by asking me to choose 1, 2, 3, or 4.

AI Prompt

GrammarShort practice

Use my profile from Part 1.
Please help me practice grammar.

Rules:
- Give me one question at a time.
- Wait for my answer before continuing.
- Keep explanations short.
- If I make a mistake, show only one correct example sentence.
- Give me only 5 questions total.

Start with one easy question.

AI Prompt

VocabularyUseful words

Please help me practice vocabulary.

The topic is travel English.

Rules:
- Give me one word at a time.
- Explain the meaning simply.
- Show one short example sentence.

- Give me only 5 words total.

Start with the first word.
Use it creatively

Change “hotel check-in” to “restaurant,” “airport,” “job interview,” “presentation practice,” or anything else. If you are not sure what to practice, ask the AI: “What kind of English practice can you do with me?”
